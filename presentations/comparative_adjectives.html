Comparative Adjectives

🌟 Why do we compare things?

Instead of: "This apple is big. That apple is big too."
We say: "This apple is bigger than that apple!"

📝
➕

📝 Short Words

Just add -er

📏 Rule for Short Words (1-2 syllables)

Add -er to the end of the word

This is the most common way to make comparatives!

✨ Word Transformations

tall → taller
fast → faster
old → older
young → younger

🏃 Simple -er words

  • small → smaller
  • cold → colder
  • long → longer
  • short → shorter
  • clean → cleaner
  • cheap → cheaper

⚡ Special spelling rules

  • big → bigger (double g)
  • fat → fatter (double t)
  • hot → hotter (double t)
  • easy → easier (y → i)
  • happy → happier (y → i)
  • lazy → lazier (y → i)
My brother is taller than me. → He has more height.
This car is faster than that one. → This car can reach 200 km/h, that one only 150 km/h.
Summer is hotter than winter. → It has more temperature.
📚
➕

📚 Long Words

Add "more" before the word

📏 Rule for Long Words (3+ syllables)

Put "more" in front of the word

Keep the original word unchanged!

✨ Word Transformations

beautiful → more beautiful
expensive → more expensive
interesting → more interesting
comfortable → more comfortable

🎨 3-syllable words

  • beautiful → more beautiful
  • dangerous → more dangerous
  • popular → more popular
  • difficult → more difficult

📖 4+ syllable words

  • comfortable → more comfortable
  • intelligent → more intelligent
  • incredible → more incredible
  • fascinating → more fascinating
  • careful → more careful
  • modern → more modern

🎯 Important tip:

❌ NEVER say: "more prettier" or "beautifuler"
✅ ALWAYS say: "prettier" or "more beautiful"

This painting is more beautiful than that one.
Mathematics is more difficult than English for me.
The new phone is more expensive than the old one.
🔄
⚡

🔄 Special Words

Irregular Comparatives

🎭 Rule for Irregular Words

These words change completely!

You need to memorize these special forms

✨ Special Transformations

good → better
bad → worse
far → farther/further
little → less

⭐ Most Common Irregular

  • good → better
  • bad → worse
  • far → farther/further
  • little → less
  • much/many → more

🎯 Memory Tips

  • good → better (like "best")
  • bad → worse (like "worst")
  • far → farther (physical distance)
  • far → further (abstract concepts)
  • far → further (ideas)
  • little → less (amount)
This pizza is better than yesterday's pizza.
Today's weather is worse than yesterday.
I need less sugar in my coffee.

🚨 Common Mistakes:

❌ "gooder", "badder", "more good"
✅ "better", "worse"

📝 Spelling Rules

Special spelling changes

👥 Double the consonant

One vowel + one consonant → double the consonant + er

  • big → bigger
  • fat → fatter
  • hot → hotter
  • sad → sadder
  • thin → thinner

🔄 Change y to i

Consonant + y → consonant + ier

  • happy → happier
  • easy → easier
  • lazy → lazier
  • busy → busier
  • funny → funnier

✨ Drop silent e

Words ending in silent 'e' → drop 'e' + er

  • nice → nicer
  • large → larger
  • simple → simpler
  • brave → braver
  • late → later
  • wide → wider

🔍 How to know which rule to use?

1. Count syllables

1-2 syllables = usually add -er

3+ syllables = usually add "more"

2. Look at the ending

Consonant + y = change to -ier

Short vowel + consonant = double + er

3. Check if irregular

good, bad, far, little = special forms

When in doubt, check a dictionary!

💡 Practice tip:

Try saying both forms out loud:
"more happy" vs "happier" - which sounds better?
Your ear will often guide you to the right answer!
